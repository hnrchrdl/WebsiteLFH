== Description ==

Bootstrap img-responsive - Automatically add img-responsive class to all post and page content.

If you like it then please find the time to rate it.

== Installation ==

* <em>Step 1:</em> Upload `img-responsive` folder to the `/wp-content/plugins/` directory or click `Install` within the Add new plugin section.
* <em>Step 2:</em> Activate the plugin through the 'Plugins' menu inside the WordPress admin.

All of your post and page content will now have img-responsive class attached to images.
